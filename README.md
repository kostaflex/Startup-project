Overview:
This project involves a comprehensive End-to-End Data Analysis of a global startup dataset (Crunchbase) to identify investment patterns and sector-specific innovations over the last decade. As a Physics student transitioning into Data Analytics, I applied rigorous statistical methods to transform "noisy" raw data into actionable business insights.

Key Features & Workflow:

Data Wrangling & Cleaning: Managed a dataset of +10,000 records using Pandas. Implemented advanced cleaning techniques, including handling high-percentage null values, standardizing inconsistent date formats, and converting currency strings into numerical data.

Exploratory Data Analysis (EDA): Conducted a deep dive into funding rounds to identify the most promising sectors. 

Statistical Insights: Applied data normalization and outlier detection to differentiate between market "noise" and genuine emerging trends in sectors like AI, FinTech, and AgTech.

Tech Stack:

Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn.

Environment: Deepnote.
